# Gestor de Tareas en Consola

## Descripción
Este proyecto es un gestor de tareas simple que funciona en la consola.
Permite al usuario crear tareas, verlas, marcarlas como realizadas y eliminarlas.

El objetivo del proyecto es practicar lógica básica de programación en Python.

## Funcionalidades
- Agregar tareas
- Ver la lista de tareas
- Marcar tareas como realizadas
- Eliminar tareas

## Tecnologías utilizadas
- Python
- Terminal / Consola

## Cómo funciona
El programa muestra un menú en consola con distintas opciones.
El usuario elige una opción y el sistema ejecuta la acción correspondiente.

Las tareas se guardan en una lista de diccionarios mientras el programa está en ejecución.

## Estructura del proyecto
- tareas.py: controla el menú y el flujo principal del programa
- utils.py: contiene las funciones que trabajan con las tareas

## Aprendizajes
- Uso de listas y diccionarios
- Funciones
- Condicionales y ciclos
- Separación de lógica en distintos archivos
- Pensamiento lógico para resolver problemas

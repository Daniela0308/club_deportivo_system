# Club Deportivo Management System

Sistema de gestión para un club deportivo.

## Objetivo
Desarrollar un sistema completo para administrar jugadores, pagos, roles y control interno de un club deportivo, aplicando buenas prácticas de programación y arquitectura de software.

## Tecnologías
- Python
- Git & GitHub
- SQLite (más adelante)
- FastAPI (más adelante)
- Frontend web (más adelante)

## Estado del proyecto
En desarrollo – desde fundamentos hasta sistema completo.
